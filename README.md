# Portfolio Performance & Trend Analysis (Power BI)

## 📌 Overview
This project analyzes a multi-asset investment portfolio using Power BI dashboards to track performance, returns, and allocation trends.

## 🎯 Objectives
- Evaluate portfolio performance
- Analyze asset allocation
- Track growth trends over time
- Generate decision-making insights

## 🛠 Tools Used
- Power BI
- DAX
- Data Modeling
- Excel (Data Source)

## 📊 Dashboard Features
- Portfolio Growth Trend
- Asset Allocation Breakdown
- Return Analysis
- Interactive Filters (Slicers)

## 🔍 Key Insights
- Identified top-performing assets
- Visualized portfolio growth over time
- Highlighted allocation inefficiencies
- Enabled data-driven investment decisions

## 🚀 Outcome
This project demonstrates dashboarding, data visualization, and analytical thinking skills required for data analyst roles.
